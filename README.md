# 🌦️ Weather Report Dashboard — Power BI

A live weather dashboard built in Power BI Desktop using WeatherAPI.com

## Preview


## Features
- Real-time weather for 6 Indian cities
- 7-day forecast with dynamic icons
- AQI with color-coded categories
- Temperature trend & rainfall charts
- Sunrise/Sunset, UV Index, Key Highlights

## Tools Used
- Microsoft Power BI Desktop
- WeatherAPI.com REST API
- DAX (SELECTEDVALUE, SWITCH, CALCULATE, FIRSTDATE)
- Power Query M

## Setup
1. Get a free API key from weatherapi.com
2. Open weatherapi.pbix in Power BI Desktop
3. Go to Transform Data → replace API key in all 6 queries
4. Click Refresh
